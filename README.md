# Installation

``` bash
$ npm install -g dme-cli
```

# Usage

``` bash
$ dme init <project-name>
```

# Quick start

``` bash
# init a project with dme-template
dme init my-project
cd my-project
# or cnpm install
npm install
npm run dev
```
